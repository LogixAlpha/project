## To write a paragraph just type directly :

- This is a paragraph. To create a line break or new line , just leave two or more space in the end of the paragraph , then press ENTER.  

- This is a new line 

## To **BOLD** a word just type ** word ** (without space)

- We can use also __ word __ (without space) to bold.  

- Additionaly , we can also __bold__ a word in the middle of sentence like this : Mark**d**own.

- I prefer using ** to **bold** than using __.

## Move on to *Italic* :

- To make a word *Italic*, we put it between **one** "*" like * word * (without space)  
**example**: *word*

- We can also italicize a word in  the middle of sentence  

- for **example** : Mark*d*own.

## This one in interesting : **Bold** and *Italic* at the same time:

- To make a word ***Bold and Italic*** , we put it between **three** (***) like *** word *** (without space). 

- **example** : ***Markdown*** 

## Blockquotes using ">" :

> This is a Blockquote.  

- To exit the blockquote just press Enter 2 time [ ***that what i thought*** ]

- We can also type 2 blockquotes or plus named *Nested Blockquote* 

**example** : 

> This is the first blockquote.
>> This is the second blockquote.
>>> This is the third blockquote.
>>>> This is the fourth blockquote.

- We can also Insert elements into blockquotes 

**example** :  

> This is a **Bold** text .
>> This is a *Italic* text.
>>> This is a ***Bold + Italic*** text.


## Lists : 
### Ordered List : 

- To Create an **ordered** list , we simply type the number with a period after it , like *1. / 2. / 3. / ..*  , It is not necessary to follow to order we can just type the same number : *1. / 1. / 1. / ..*   
**example** :  
1. First item .
1. Second item .
    1. Category 1
    2. Category 2
1. Third Item.  

### Unordered List : 

 To create an **unordered** list , we usually type -- followed by a word .  
**example** :  
- First thing.
- Second thing.
    - First category
    - Second Category
- Third thing.  

To write a number followed by a period "." on an **unordered** list , Devs prefer typing like this :   

- 2025\. It was a good year.

## Links : 

To create a link, enclose the link text in brackets [Duck Duck Go] and then follow it immediately with the URL in parentheses (https://duckduckgo.com).
***
**example** :  [ Don't click me!](https://www.google.com)
***
### Adding Titles :

You can optionally add a title for a link. This will appear as a tooltip when the user hovers over the link. To add a title, enclose it in quotation marks after the URL. 
**example** : [Look here](file.html "HTML file")
